# 백두산 화산재난 정보 웹사이트

이 프로젝트는 백두산 폭발 관련 재난 정보를 제공하기 위한 단일 페이지 웹사이트 예시입니다.

## 포함 요소
- 현재 재난 단계
- 백두산 화산 활동 현황
- 긴급 공지사항
- 대피 안내
- 폭발 시 화산재 이동 경로
- 풍향 풍속 정보
- 피해 예상 지역
- 피해 신고
- 교육 자료
- 링크 연결
- 시민 행동 요령 안내문, 주변 대피소, 시민안전보험, GPR 탐사지도
- 실시간 재난 정보
- 재난 지원 및 피해보상 안내

## 실행 방법
1. `index.html` 파일을 브라우저에서 열어 확인합니다.
2. 향후 데이터는 JavaScript 또는 백엔드 API로 확장할 수 있습니다.
